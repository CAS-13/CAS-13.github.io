Portfolio

 

U.S. Hospitals

This web map tool allows you to search for hospitals in the United States that meet user-specified criteria.

Data Source: Hospitals feature layer, Homeland Infrastructure Foundation-Level Data (HIFLD). Accessed January 6, 2024. Closed hospitals are filtered out of the dataset.

Glacial Landsystems

This webpage uses ArcGIS Maps SDK for JavaScript to display a REST feature layer of glacial landsystems in Michigan. The symbology, legend, and pop-ups were configured programatically.

World Cities

This webpage displays populations for major world cities based on a continent selected by the user. All features and functionality were configured programatically.

Motor Vehicle Incidents (Esri)

This web application uses ArcGIS Maps SDK for JavaScript to show the impacts of traffic violence involving bicyclists and pedestrians in the City of St. Louis, Missouri. Community-reported incidents were collected using Survey123.

Motor Vehicle Incidents (FOSS)

This web application uses all open source tools, including Leaflet, QGIS, and GeoPandas, to show the impacts of traffic violence involving bicyclists and pedestrians in the City of St. Louis, Missouri.
